# Algorithms-SET-3
# Monte-Carlo: пересечение 3 кругов

## Author: Капогузов Максим, БПИ246

### Репозиторий для задачи: оценка площади пересечения трёх кругов методом Монте-Карло с двумя прямоугольниками (wide/small) и сравнение с точным значением

---

Файлы:
- main.cpp - C++ генератор эксперимента; выводит CSV со столбцами
N, S_est_wide, rel_err_wide, S_est_small, rel_err_small
- input.csv - сырые результаты экспериментов.
- plot.py - построение графиков; сохраняет area_vs_N.png и relerr_vs_N.png.
- output.png - график: оценка площади vs N (+ линия S_true).
- README.md - краткие выводы и описание
